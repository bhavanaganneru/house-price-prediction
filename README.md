# House Price Prediction

This project predicts house prices using Machine Learning.

## Tools Used
- Python
- Jupyter Notebook
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn

## Steps
1. Data preprocessing
2. Feature selection
3. Model training
4. Prediction

## How to Run
Open `house_price.ipynb` in Jupyter Notebook and run all cells.
